# Installation guide
	- copy vào thư mục solr.solr.home, sửa cấu hình sqlserver trong file conf/data-sources.xml
	- bật lại solr
	- Muốn chạy import thì vào handler: /import?cmd=import
	- Muốn xem trạng thái: /import?cmd=status
